Vimium 是浏览器的一个插件, 可以使用 Vim 的键位操作浏览器, 减少鼠标的使用.

下面主要列出个人常用的快捷键便于查阅.

# 页面操作

* `?`: 显示帮助对话框获取所有可用快捷键列表.
* `h/j/k/l`: 方向键, 滚动范围小.
* `d/u`: 向下/上滚动半页.
* `gg/G`: 滚动到页面顶部/底部.
* `f/F`: 打开元素定位器, 在当前标签页/新标签页打开.
* `gi`: 聚焦页面上的第一个文本输入框.
* `H/L`: 回到前一页/后一页.
* `yy`: 复制当前网址到剪贴板.
* `yf`: 复制选中的当前页面中存在的 URL 链接到剪贴板.
* `v`: 进入预览模式, 使用 `p/P` 粘贴, `y` 拷贝.
* `ma/A`: 设置本地/全局标记 `a`/`A`.
* `r`: 刷新页面.
* `gs`: 查看源码.

## 新页面

* `o/O`: 从 URL, 书签, 历史记录中搜索地址, 回车在当前标签页/新标签页打开.
* `b/B`: 仅从书签搜索地址, 回车在当前标签页/新标签页打开.
* `T`: 搜索当前浏览器的所有标签.

## 搜索

* `/`: 进入查找模式, 按 `<Enter>` 键搜索, 或按 `<Esc>` 键取消.
* `n/N`: 查找下一个/上一个匹配项.

# 标签操作

* `J/K`: 跳到左标签/右标签.
* `g0/g$`: 跳转到第一个/最后一个标签, 也可以换不同的数字进行跳转.
* `t`: 创建一个新标签.
* `x`: 关闭当前标签.
* `p/P`: 在当前标签页/新标签页打开剪切板中的 URL, 如果不是 URL 则使用默认搜索引擎搜索.
