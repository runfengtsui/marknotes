# 快捷键

WPS 可以通过 `Alt` 键来选择使用各种编辑功能. 即按下 `Alt` 键后, WPS 上方菜单栏每一项都会出现一个字母供选择, 敲击不同的字母使用不同的功能或者出现下一层的选择(`Esc` 键可以返回上一层次的选择). 这种操作可以解放鼠标的使用.

# 数学公式的输入

首先, 快捷打开公式编辑器: `Alt` + `I` + `Q`; 相应的公式编辑器界面使用 `Alt` + `F` + `X` 快捷键可以推出公式编辑器界面返回 WPS 的界面.

其次, 常见的公式位以及特殊符号输入的快捷键:

* 分式 (Fraction): `Ctrl` + `F`;
* 根号 (Root): `Ctrl` + `Q`;
* 定积分 (Integral): `Ctrl` + `I`;
* 求导符号: `Ctrl` + `Alt` + `'`;
* 上标: `Ctrl` + `H`;
* 下标: `Ctrl` + `L`;
* 上下标: `Ctrl` + `J`;
* 圆括号: `Ctrl` + `9`;
* 方括号: `Ctrl` + `[`;
* 花括号: `Ctrl` + `Shift` + `[`;
* 希腊字母: `Ctrl` + `G`, `a`;

以上快捷键中的各种括号是成对存在的, 而且这种括号可以根据括号内的内容自动改变高度大小.

对于希腊字母的输入, `Ctrl` + `G` 快捷键是进入输入希腊字母的模式, 此时按小写字母输入的是对应的希腊字母, 对应法则应该是希腊字母的英文首字母. 相应的, 输入大写字母得到的是大写希腊字母.
