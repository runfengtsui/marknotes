# 高等代数解题方法总结
## 非异矩阵的判定方法

根据谢启鸿教授博客的 [复旦大学数学学院19级高等代数I期中考试第七大题的三种证法及其推广](https://www.cnblogs.com/torsor/p/12272213.html) 这一篇博文，判定非异矩阵的常见方法有五种，分别是：

1. 行列式的计算；
2. 凑因子法；
3. 线性方程组求解理论的应用；
4. 互素多项式的应用；
5. 特征值的计算。

## 可对角化的判定准则

一般 $n$ 维复线性空间 $V$ 上的线性变换 $\varphi$(或 $n$ 阶复矩阵 $A$) 可对角化，通常有以下六种方法：

1. $\varphi$ 可对角化的充要条件是 $\varphi$ 有 $n$ 个线性无关的特征向量；
2. 若 $\varphi$ 有 $n$ 个不同的特征值，则 $\varphi$ 可对角化；
3. $\varphi$ 可对角化的充要条件是 $V$ 是 $\varphi$ 的特征子空间的直和；
4. $\varphi$ 可对角化的充要条件是 $\varphi$ 有完全的特征向量系，即对 $\varphi$ 的任一特征值，其几何重数等于其代数重数；
5. $\varphi$ 可对角化的充要条件是 $\varphi$ 的极小多项式无重根；
6. $\varphi$ 可对角化的充要条件是 $\varphi$ 的 Jordan 块都是一阶的，或等价地，$\varphi$ 的初等因子都是一次多项式。

谢启鸿教授在 [可对角化的其他判定准则及其应用](https://www.cnblogs.com/torsor/p/13569961.html) 这一篇博文中，除了上述常见的六种方法，还推广得到了其他判定准则，可以阅读之。

## 矩阵秩的估计

谢启鸿教授的 [复旦大学数学学院19级高等代数I期中考试第六大题的三种证法及其延拓](https://zhuanlan.zhihu.com/p/270574919) 这一篇博文提到矩阵秩的估计通常有以下三种方法，分别是：

1. 从矩阵秩的基本等式和不等式出发, 利用矩阵的初等变换来处理；
2. 利用线性方程组的求解理论来处理；
3. 利用线性空间理论来处理。
